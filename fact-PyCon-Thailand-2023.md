ในงาน PyCon Thailand 2023 (Lightning Talk) นาทีที่ 02:43 นายโบ๊ทได้นำเสนอสไลด์ที่มีรูปคู่กับบุคคลอื่น และระบุข้อความบนสไลด์ว่า "between bf and gf" พร้อมพูดบรรยายว่าบุคคลในภาพคือ "My girlfriend"
แม้ในช่วงท้ายจะมีการกล่าวว่า "Just Kidding" (ล้อเล่น) เกี่ยวกับเรื่องราวการทะเลาะกันเรื่อง Code Editor แต่เจตนาของการนำเสนอทำให้ผู้รับชมเข้าใจผิดว่าบุคคลทั้งสองมีความสัมพันธ์กันจริง
ข้อเท็จจริง: ความสัมพันธ์ดังกล่าวไม่มีอยู่จริง เป็นการนำภาพและข้อมูลส่วนตัวของบุคคลอื่นมาผูกเรื่องราวเท็จ (Fabrication) เพื่อใช้ประกอบการบรรยาย ซึ่งสอดคล้องกับพฤติกรรมชอบแอบอ้างเรื่องส่วนตัวตามที่ระบุไว้ใน Repo
